# PocketLedger Monetization Strategy

## Core Philosophy
We will never hold user financial tracking hostage behind a subscription. Core entry, editing, and local tracking is free forever.

## Freemium Tier
- **Free**: Unlimited manual transactions. 5 Categories. 3 Budgets. 1 Month analytics history.
- **Lifetime Premium ($15)**: 
  - Unlimited Categories & Budgets
  - Historical Analytics (All Time)
  - CSV Export/Import
  - Local Encrypted JSON Backups
  - Dark Mode & Custom Accents

This avoids the fatigue of "another subscription" while maintaining high conversion rates.
