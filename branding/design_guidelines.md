# Design Guidelines

## Typography
- Utilize modern sans-serifs (Inter or Roboto) to keep numeric data highly legible.
- Numbers must always use tabular lining to ensure vertical decimal alignment on lists.

## Interface Elements
- **Cards**: Flat with soft 8px to 12px radiuses. Minimal drop shadows, using border strokes (e.g. 1px solid #E2E8F0) to maintain a fast, offline appearance.
- **Micro-interactions**: Scale down by 2% on button presses instead of ripples.
- **Charts**: Use FL Chart. Lines should be smooth (curved), with gradient fills below the trend line pointing towards the floor.
- **Icons**: Utilize bold, filled icons for active states and outline icons for restful states. The icon family should maintain rounded edges to remain friendly.
